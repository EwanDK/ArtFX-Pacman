Développé avec Raylib.

-Les déplacements ne se font qu'avec les flèches directionnelles
-Les fantômes sont statiques, leurs tests de collisions avec les murs font planter le programme
-Erreurs dans la logique de victoire/défaite, au lieu d'afficher le texte le programme gèle
